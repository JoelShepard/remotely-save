# Install Remote Sync via BRAT

[BRAT](https://obsidian.md/plugins?id=obsidian42-brat) (Beta Reviewer's Auto-update Tool) lets you install Obsidian plugins directly from GitHub repos, bypassing the official community store.

## Steps

1. Open Obsidian and go to **Settings → Community Plugins**
2. Search for and install **BRAT** (id: `obsidian42-brat`)
3. Once installed, go to **Settings → BRAT**
4. Click **Add Beta plugin**
5. Enter the GitHub repository: `JoelShepard/remotely-save`
6. Click **Add Plugin** — BRAT will download the latest release
7. Go to **Settings → Community Plugins**, refresh the list
8. Find **Remote Sync** and click **Enable**

## Updating

BRAT automatically checks for updates. You can also manually trigger an update from BRAT's settings by clicking the refresh icon next to Remote Sync.

## Removing

Either disable it in Community Plugins, or use BRAT's "Delete" option to fully remove it.
